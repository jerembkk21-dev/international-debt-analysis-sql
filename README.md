# international-debt-analysis-sql
Analyzed World Bank international debt data using SQL to identify the number of indebted countries, determine the country with the highest total debt, and find the country with the lowest principal repayment. This project demonstrates the use of aggregation, grouping, filtering, and sorting to answer real-world economic questions.

# International Debt Analysis (SQL)

This project analyzes international debt data from the World Bank using SQL.
It answers key questions about:
- Number of countries in debt
- Country with the highest total debt
- Country with the lowest principal repayments

## Dataset
Source: World Bank (via DataCamp)
Table: `international_debt`

## Skills Demonstrated
- SQL aggregation (SUM, MIN, COUNT)
- GROUP BY and ORDER BY
- Filtering with WHERE
- Data analysis logic

## Platform
This project was completed on DataCamp.
The database environment is read-only.

## Key Results
- Total distinct countries: 124
- Highest debt country: 285793494734.2
- Lowest principal repayment country: 825000
